## 实验目标

实现 MVP 变换矩阵（模型 - 视图 - 投影），将 3D 三角形变换到 2D 屏幕并显示。
---
## 核心原理
### 图形流水线变换链条

课堂上曾经提过，模型局部坐标系 → 模型变换 → 世界坐标系 → 视点变换 → 眼睛坐标系 → 投影变换 → 图像坐标系

### 齐次坐标

所以使用 4 维向量表示 3D 点 `(x, y, z, 1)`，使得所有变换都能用 4×4 矩阵统一表示。

### 三个核心变换

| 变换 | 作用 | 矩阵形式 |
|------|------|----------|
| **模型变换** | 绕 Z 轴旋转（刚体变换） | 2D 旋转矩阵扩展到 4×4 |
| **视图变换** | 将相机移到世界原点 | 平移矩阵，平移量 = -eye_pos |
| **投影变换** | 透视投影（近大远小） | 挤压矩阵 × 正交矩阵 |

### 透视投影流程

1. **透视挤压**：将平截头体（金字塔形）压成正交长方体
2. **正交投影**：将长方体映射到 [-1, 1]³ 的标准设备坐标
3. **透视除法**：齐次坐标除以 w 分量，实现近大远小效果

### MVP 组合

最终变换矩阵为三个矩阵的乘积（从右往左执行）：

$$
MVP = M_{proj} \times M_{view} \times M_{model}
$$
